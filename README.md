# xchgr
XCHG router service

This is HTTP-server. It uses port 8084 (HTTP 1.1 without SSL).

## API
### Write Frames
```
/api/w
```
### Read Frames
```
/api/r
```
### Resolve xchg Domain Name
```
/api/ns
```
### Get Debug Information
```
/api/debug
```
No parameters. It returns JSON.
### Get Statistics
```
/api/stat
```
No parameters. It returns JSON.
